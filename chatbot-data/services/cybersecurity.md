- **Overview**: Enforce regulations across AWS, Azure, and GCP public clouds. 
- **Frameworks Supported**: CIS benchmarks, SOC 2, ISO 27001, HIPAA, PCI-DSS, GDPR.

- **Overview**: Build secure base Docker/Kubernetes container images.
- **Actions**: Scan for vulnerabilities (CVEs), disable root privileges, and strip out unnecessary packages to reduce attack surface.

## Sensor: Runtime Protection

- **Actions**: Detect runtime exploits, unauthorized shell executions, zero-day behaviors, and configuration drift.

## Security Compliance & Audit
